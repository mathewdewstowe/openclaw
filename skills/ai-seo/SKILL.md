# AI SEO

Help the user get cited by AI systems. Focus on extractability, authority, freshness, and source presence.

## First moves

1. Check for `.agents/product-marketing-context.md` or `.claude/product-marketing-context.md`.
2. Clarify the goal:
   - improve AI citations
   - appear in AI Overviews
   - beat competitors in AI answers
   - optimize existing pages vs create new ones
3. Identify the priority queries and pages.

## Audit workflow

### 1. Check citation visibility
Test the most important queries across Google AI Overviews, ChatGPT, and Perplexity.
Track:
- whether the brand is cited
- which competitors are cited
- what page types are winning

### 2. Check extractability
For each priority page, verify:
- clear answer in the opening section
- scannable headings matching query intent
- self-contained answer blocks
- tables/lists where useful
- FAQs where appropriate
- visible author/expert attribution
- recent update date when relevant

### 3. Check authority
Look for:
- original data
- stats with sources
- expert quotes
- clear point of view
- consistent brand claims across the site and third-party sources

### 4. Check crawler access
Review `robots.txt` for AI-relevant crawlers such as:
- GPTBot / ChatGPT-User
- PerplexityBot
- ClaudeBot / anthropic-ai
- Google-Extended
- Bingbot

If blocked, explain the tradeoff between training restrictions and citation visibility.

## Prioritization

Recommend fixes in this order:
1. pages already close to ranking/citation
2. high-intent comparison or category pages
3. pages with strong expertise but poor structure
4. missing third-party proof or mentions

## Core recommendations

### Make content easier to quote
- answer the main question early
- keep definitions crisp
- use descriptive subheads
- add concise comparison tables
- include short, quotable summary blocks

### Make content easier to trust
- cite sources
- show authorship and expertise
- keep claims consistent across the web
- publish or reference original evidence where possible

### Make content easier to find
- strengthen internal links to priority pages
- align title/H1/subheads to real query phrasing
- keep pages fresh when facts change
- support with schema where relevant

## Deliverables

When helpful, produce one or more of:
- AI visibility audit
- prioritized fix list
- rewritten page outline
- AI-citation-friendly content brief
- competitor gap summary
